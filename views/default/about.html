Happy Goblin

Happy Goblin is a hobby project to collect and manage Magic the Gathering game stats for your playgroup. Happy Goblin is very much a work in progress.

Logins

Happy Goblin uses the Mozilla Persona service to manage user logins. It just makes it easier for all of us. No there are no local only accounts for Happy Goblin

Data

As this is a hobby project, I can't guarantee data backups. Of course I do my best to not lose any data, but with the current resouces I cannot guarantee it (e.g. server failure).

The user data might be used for collecting stats, but not for selling. That's just mean.

API and Data export

Currently there is no end-user reachable API for Happy Goblin. But a web app without a REST API is just plain silly, so it's on the todo list.

Mobile Apps

There are no mobile apps at the moment. They are a consideration for the future

Liability and stuff

Yeah, use at your own risk. I might accidentally screw things up.
